## API

| Método | Url         | Descrição            |
| ------ | ----------- | -------------------- |
| GET    | /users      | Lista de Usuários    |
| GET    | /users/{id} | Busca por id         |
| POST   | /users      | Cria um novo usuário |
| PUT    | /users/{id} | Atualiza o usuário   |
| DELETE | /users/{id} | Exclui o usuário     |
